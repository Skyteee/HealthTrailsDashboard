API Sources

OpenWeather API

Description: Provides real-time weather data, forecasts, and alerts.

Usage: Used for weather overlays and temperature data on maps.

Pricing: Free for basic usage, tiered pricing for advanced features.

Example Call:

GET https://api.openweathermap.org/data/2.5/weather?q=Berlin&appid=YOUR_API_KEY

Example Response: JSON data with temperature, humidity, and weather conditions.

OpenStreetMap API

Description: Open-source map data with detailed geographic information.

Usage: Utilized for mapping paths, trails, and other geographic features.

Pricing: Free with an open-source license.

Example Call:

GET https://nominatim.openstreetmap.org/search?q=Berlin&format=json

Example Response: JSON with latitude, longitude, and place details.
